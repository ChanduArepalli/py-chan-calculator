# Py-Chan-Calculator ![PyPI](https://img.shields.io/pypi/v/py-chan-calculator) 
This is a simple calculator package

### How to install
```shell
pip install pychancalculator
```

### How to import
```pycon
from pychancalculator.calculator import Calculator
```

### Code
```python
# create an instance
cal = Calculator()

# For the Addition
result = cal.add(2,3)
print(result)
# return 5 

# For the Subtraction
result = cal.dif(5,3)
print(result)
# return 2

# For the division
result = cal.div(6,3)
print(result)
# return 2 

# For the multiplication
result = cal.multi(8,2)
print(result)
# return 4 
```
